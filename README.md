# Banking Microservices Platform

Sistema bancario desarrollado con arquitectura de microservicios.

## Tecnologías

- Java 17
- Spring Boot
- Spring Cloud
- MongoDB
- Maven
- Lombok
- Eureka
- Config Server
- API Gateway

## Arquitectura

Microservicios:

- config-server
- eureka-server
- api-gateway
- customer-service
- account-service
- credit-service
- transaction-service

## Ejecución

1. Levantar Config Server
2. Levantar Eureka Server
3. Levantar API Gateway
4. Levantar microservicios
